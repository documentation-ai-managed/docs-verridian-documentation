## Core Offerings

Explore Verridian's key technologies through these focused areas.

<Tabs>
  <Tab title="AI Inference" icon="cpu">
    Run transformer models like Llama or Stable Diffusion in-browser using optimized WebGPU backends.
    
    ```javascript
    import { VerridianAI } from '@verridian/ai-sdk';
    
    const ai = new VerridianAI({ model: 'llama-7b' });
    const result = await ai.infer('Explain quantum computing.');
    console.log(result.text);
    ```
  </Tab>
  <Tab title="WebGPU Rendering" icon="monitor">
    Accelerate 3D graphics and compute shaders for games, visualizations, and simulations.
    
    ```javascript
    import { VerridianGPU } from '@verridian/webgpu-sdk';
    
    const gpu = new VerridianGPU();
    await gpu.init();
    const shader = gpu.createComputeShader('my-shader.wgsl');
    gpu.dispatch(shader, { workgroupSize: [64, 1, 1] });
    ```
  </Tab>
</Tabs>

## Get Started in Minutes

Follow these steps to launch your first Verridian project.

<Steps>
  <Step title="Install SDK" icon="download">
    Add the core package to your project.
    
    <CodeGroup tabs="npm,yarn,pnpm">
      ```bash
      npm install @verridian/sdk
      ```
      ```bash
      yarn add @verridian/sdk
      ```
      ```bash
      pnpm add @verridian/sdk
      ```
    </CodeGroup>
  </Step>
  <Step title="Initialize Client" icon="settings">
    Create a client instance with your API key.
    
    ```javascript
    import { VerridianClient } from '@verridian/sdk';
    
    const client = new VerridianClient({
      apiKey: 'your-api-key-here'
    });
    ```
  </Step>
  <Step title="Run First Compute" icon="rocket">
    Execute a simple WebGPU kernel.
    
    ```javascript
    const result = await client.compute({
      shader: 'vec4 main() { return vec4(1.0); }',
      size: 1024
    });
    console.log('Compute complete:', result);
    ```
  </Step>
</Steps>

<Expandable title="Advanced Configuration" default-open="false">
  Customize your setup for production:
  
  ```javascript
  const client = new VerridianClient({
    apiKey: 'prod-key',
    gpu: { adapter: 'high-performance' },
    ai: { maxTokens: 4096 }
  });
  ```
</Expandable>
