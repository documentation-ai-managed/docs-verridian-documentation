Verridian releases follow semantic versioning. You receive notifications in the dashboard. Check the tags below for impact on your workflows.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "improvement"]}>

## New Features

- Added real-time AI collaboration sessions supporting up to `>50` concurrent users.
- Introduced WebGPU shader caching for `<20%` reduced load times.

## Improvements

- Optimized tensor operations by `40%` using native WebGPU compute shaders.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  // New collaboration API
  const session = await verridian.collab.create({
    model: 'gpt-4o',
    maxUsers: 50
  });
  ```
  ```python
  # New collaboration API
  session = verridian.collab.create(
      model='gpt-4o',
      max_users=50
  )
  ```
</CodeGroup>

## Bug Fixes

- Fixed memory leaks in long-running GPU sessions.
- Resolved shader compilation errors on mobile WebGPU.

</Update>

## Upgrade Instructions

Follow these steps to update your Verridian installation across platforms.

<Tabs>
  <Tab title="Node.js" icon="nodejs">
    <Steps>
      <Step title="Update Package" icon="package">
        Run the upgrade command.
        ```bash
        npm update verridian@latest
        ```
      </Step>
      <Step title="Rebuild Shaders" icon="code">
        Clear cache and rebuild.
        ```bash
        verridian shaders rebuild
        ```
      </Step>
      <Step title="Test Deployment" icon="play">
        Verify in staging environment.
      </Step>
    </Steps>
  </Tab>
  <Tab title="Python" icon="python">
    <Steps>
      <Step title="Update Package" icon="package">
        ```bash
        pip install --upgrade verridian
        ```
      </Step>
      <Step title="Reconfigure" icon="settings">
        Update `config.yaml` with new endpoints.
      </Step>
    </Steps>
  </Tab>
</Tabs>
